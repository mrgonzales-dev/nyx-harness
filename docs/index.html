Nyx is an AI harness built with documentation. Use it when you're stuck on syntax, when you want to understand how a language works, or when you just want to learn. You write the code — Nyx is your guide, not your agent.

Local & Private

Docs On Demand

Install any DevDocs docset from the terminal. /docs install go — it's that fast.

Code Mode

Toggle /code for deterministic code-only output. Temperature forced to 0.0, only fenced blocks survive.

Context-Aware

Select any text in a doc and feed it to the AI with a. Follow up with /followup to dig deeper.
